## ğŸ“¦ Installation

### macOS
1. Download `Timer_0.1.0_aarch64.dmg` (Apple Silicon) or `Timer_0.1.0_x64.dmg` (Intel)
2. Open the DMG file
3. Drag Timer.app to Applications folder

### Windows
1. Download `Timer_0.1.0_x64-setup.exe`
2. Run the installer
3. Follow the installation wizard
4. Launch Timer from Start Menu

1. Download `Timer_0.1.0_amd64.AppImage`
2. Make it executable: `chmod +x Timer_0.1.0_amd64.AppImage`
3. Run: `./Timer_0.1.0_amd64.AppImage`
